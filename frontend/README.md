# Frontend - Maritime Transportation Planner

Folder ini berisi kode frontend untuk aplikasi Perencanaan Transportasi Laut dan Pelabuhan.

## Spesifikasi Teknologi
*   **Framework:** Next.js
*   **Bahasa:** TypeScript
*   **Styling:** Tailwind CSS

## Tanggung Jawab Frontend
Frontend dirancang sebagai antarmuka interaksi pengguna yang tidak melakukan kalkulasi rekayasa secara langsung. Perannya meliputi:
1. Mengumpulkan input parameter proyek dan skenario.
2. Mengirimkan instruksi eksekusi kalkulasi atau optimasi ke backend.
3. Menampilkan visualisasi hasil kalkulasi, grafik perbandingan skenario, dan penjelajah rantai kalkulasi (*traceability explorer*).
